## 🎨 Diretrizes de Design

- Layout clean (padrão Nexus Labz)
- Destaque para números grandes:
  - 3 entrevistas
  - R$20K
  - 2 semanas
- Pouco texto, alta legibilidade
- Uso de contraste para chamar atenção

---

## ⚠️ Boas Práticas

- Não poluir com texto excessivo
- Priorizar impacto visual
- Usar dados concretos sempre
- Evitar frases genéricas
